This directory contains the production font assets used by the site.

- `Cinzel-Regular.ttf`
- `Cinzel-SemiBold.ttf`
- `Cinzel-Bold.ttf`
- `Marcellus-Regular.ttf`

Source families:
- Cinzel: https://github.com/librefonts/cinzel
- Marcellus: Google Fonts / Libre source distribution

Licenses:
- `Cinzel-OFL.txt`
- `Marcellus-OFL.txt`

Notes:
- The checked-in source dumps were removed intentionally; only deployable font binaries live here.
- `Cinzel-SemiBold.ttf` is currently aliased from the available bold master because the imported source set did not include a semibold binary.
